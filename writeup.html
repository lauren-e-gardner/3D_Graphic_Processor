Traversal

Vertices on vertex: Starting at the halfedge connected to v, I traversed the surrounding edges on v using a loop that updated the HE to HE.opposite.next each iteration. Each iteration I would store the vertex connected to the current HE

Edges on vertex: Starting at the halfedge connected to v, I traversed the surround edges on v using a loop that updated the HE to HE.opposite.next each iteration. Each iteration I stored this HE

Faces on vertex: Starting at the halfedge connected to v, I traversed the surround edges on v using a loop that updated the HE to HE.opposite.next each iteration. Each iteration I stored the face connected to the current HE

Vertices on edge: I stored the current HE.vertex and the HE.opposite.vertex and returned

Faces on edge: I stored the current HE.face and the HE.opposite.face and returned Edges on Face: Starting at the halfedge connected to f, I traversed the surrounding edges on f using a loop that updated the HE to HE.next each iteration. Each iteration I would store the current HE

Faces on Face: Starting at the halfedge connected to f, I traversed the surrounding edges on f using a loop that updated the HE to HE.next each iteration. Each iteration I stored the HE.opposite.face

Edges between vertices: I traversed all of the vertices around v1 until I found the one that equalled v2. I then stored that halfedge and returned

Face Area

For each vertex on face f, I first cross multiplied the v[i] and v[i+1] and multiplied that by a scalar (sin(theta)*1/2) where theta is the angle between v1 and v2. I then add the (x, y, z) coordinates squared and squarerooted the result and added it to a total area variable. I then divided by the number of additions to the face and returned the result.

Per-vertex Normals

I scaled the normal of each face surrounding v by the area of that face and then applied it to the vertex.

Average Edge Lengths

I traversed the vertices surrounding vertex v and calculated the distance to each and averaged it.

Uniform Laplacian Smoothing

I implemented uniform lapacian smoothing by iterating through the neighboring vertices of each vertex v. Following the formula provided on the assignment page, I stored v.position and multiplied that by the number of neighbors that v had. I then added all of the neighbors positions up and subtracted that from the previous value. Finally I applied my delta and returned. For this smoothing function and others, I stored altered values in a modified array and applied the changes after.

Curvature-flow Laplacian Smoothing

Much like with uniform lapachian smoothing, I based my code on the algorithm given in the assignment specs. I first got the total weights of all the vertices and applied that to my scaledDelta variable along with the vertex v's position. After this I then added all of v's neighbooring vertices positions multiplied by the associated weights. I then subtracted this from the former variable and pushed it to my modified array to avoid changing vertices too early.

Truncate

To implement truncate, I first populated edge and verts-on-edge-position arrays with cloned versions of edges and vertice positions. I then split each edge connected to the current vertex and got the new midpoint along that. I saved one edge for last since the original vertex's position was needed in the calculation. The last vertext is just the original one moved to the new position. I then finished up by spliting the faces.
